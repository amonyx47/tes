Tvorba edukačného softvéru

  • Program je určený pre žiakov 1. stupňa ZŠ
  • Využijú ho žiaci hlavne pri fáze spoznávania samotného procesu triedenia

Program môže učiť:
  • pochopenie pojmu triedenia
  • všeobecné počítačové zručnosti
  • úvod do triediacich algoritmov
